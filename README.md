# agent-alpha

DnD4AI Player Agent – Slot **alpha**.

Charaktername wird automatisch vom Discord Bot-Token abgerufen.

## Setup

```bash
git clone --recurse-submodules https://github.com/dnd4ai/agent-alpha.git
cd agent-alpha
cp .env.example .env
# .env befüllen
pip install -r requirements.txt
python player_agent.py
```
